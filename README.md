# README

Getting Started

Clone the app

``$ git clone https://github.com/KevinBoston/Libraily.git``

``$ cd Libraily``

Install bundled gems

$ bundle install

Setup your database

`` rails db:migrate ``

Seed sample data, featuring Parcs & Rec references:

`` rails db:seed ``

Boot up Rails server:

`` rails s ``

Navigate to localhost:3000 and proceed with instructions.
